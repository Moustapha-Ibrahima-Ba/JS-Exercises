Leap Year

Write a function that determines if a year is a leap year.
Remember that a year is a leap year if it is divisible by 4 but not divisible by 100 unless it is divisible by 400.

  • isLeap(2008) => true
  • isLeap(1900) => false
  • isLeap(2006) => true

Dice Rolling

The purpose of this exercise is to create functions to simulate dice rolls.
Rolling a die corresponds, from a computer point of view, to drawing a random integer between 1 and 6 inclusive.

  1. Question 1

    2. Create a function that simulates rolling two dice and returns the sum. Use this function to display the result of a roll of two dice.

  2. Question 2

    3. Create a function that simulates rolling any number of dice given as a parameter. Use this function to display the result of rolling two dice then three dice.
